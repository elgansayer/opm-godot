URL Parameters & Custom CVars

URL Parameters

Append to the URL as query strings, e.g. ?map=dm/mohdm1&connect=host:port

com_target_gameGame selection: 0 = AA, 1 = SH, 2 = BT
mapLoad a map on startup, e.g. dm/mohdm1
connectConnect to a server on startup
relayWebSocket relay address for multiplayer
execExecute a config file on startup
serverSet to 1 to auto-exec server.cfg
dedicatedSet to 1 for dedicated server mode

OpenMoHAA Custom CVars

New CVars introduced by this port. Set via the console (~) or URL, e.g. ?r_shadows=1

r_shadows0 = classic shadow blobs (default); 1 = GPU shadows — dynamic shadow casting from the map’s sun direction (worldspawn sundirection/suncolor), entities cast and receive real-time shadows
r_dlight_shadows0 = off (default), 1 = dynamic point lights also cast shadows (expensive, requires r_shadows 1)
r_godot_rain0 = original engine rain, 1 = Godot GPU particle rain (default)
r_zfarDraw distance override: 0 = auto (default), positive value sets max render distance in engine units (~39 units per metre)
ui_dmbox_scaleFont scale for the death message box (default 1.0)
ui_gmbox_scaleFont scale for the game message box (default 1.0)
ui_console_scaleFont scale for the in-game console (default 1.0)
ui_minicon_scaleFont scale for the mini-console / notify area (default 1.0)

Audio

s_volumeMaster volume (0.0–1.0)
s_musicvolumeMusic volume (0.0–1.0)

Geometry & LOD

r_subdivisionsPatch tessellation (2–16, lower = more detail)
r_lodBiasModel LOD bias (0–2, higher = less detail)
ter_maxlodTerrain LOD (0–6, higher = coarser)
